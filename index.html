Remind

Remind is an assistive technology for carers and patients of memory-loss conditions. The system allows carers to remotely send messages and reminders to their patients via tablets installed in their homes.

To reduce data usage the system makes use of MQTT, a protocol for device-to-device communication with minimum overhead.

The project won TMT Magazine's Best Assistive Technology App 2016 award.

Web Toolbox

Some years ago the very handy website thetoolbox.cc went offline. It provided an amazing collection of free-to-use tools for developers.

Occasionally I come across free tools that I'd love to tell others about, but have nowhere to do this. It seemed like a great idea to reboot this website and get free tools to developers once again.

Personal Site

This exact website! I built it to show off my personal projects and gain some experience using GitHub Pages.

The site is stored in a GitHub repository, hosted using GitHub Pages, and delivered by CloudFlare.

The size of the site is kept to a minimum by using spritesheet images where necessary, and only external resources and fonts (served by Google Fonts via their CDN).
